糖果传递

这是一个经典的数学建模问题，通常被称为**”环形均分纸牌问题”**。

这道题初看是一个复杂的分配问题，但通过数学推导，我们可以将其转化为一个中位数问题。下面将为你详细拆解解题思路。

一、核心思路分析

1. 目标分析

我们要让每个人的糖果数最后都等于平均值 avg。

假设第 ii i 个小朋友初始有 aia_i ai 个糖果，总人数为 nn n，则：

avg=∑ainavg = \frac{\sum a_i}{n}avg=n∑ai

2. 建立变量

由于每个人只能给左右两边传递糖果，我们定义 xix_i xi：

  • xix_i xi 表示第 ii i 个小朋友给第 i+1i+1 i+1 个小朋友的糖果数量；
  • 如果 xi>0x_i > 0 xi>0，表示 ii i 给了 i+1i+1 i+1 糖果；
  • 如果 xi<0x_i < 0 xi<0，表示 i+1i+1 i+1 给了 ii i 糖果（负负得正）；
  • 特别地，xnx_n xn 表示第 nn n 个小朋友给第 11 1 个小朋友的数量（因为是圆圈）。

3. 列出方程

对于每个小朋友，他最后的糖果数应该等于 avgavg avg：

第 1 个人：a1−x1+xn=avg⇒x1=xn+a1−avg\text{第 1 个人：} a_1 - x_1 + x_n = avg \Rightarrow x_1 = x_n + a_1 - avg第 1 个人：a1−x1+xn=avg⇒x1=xn+a1−avg第 2 个人：a2−x2+x1=avg⇒x2=x1+a2−avg=xn+(a1+a2−2avg)\text{第 2 个人：} a_2 - x_2 + x_1 = avg \Rightarrow x_2 = x_1 + a_2 - avg = x_n + (a_1 + a_2 - 2avg)第 2 个人：a2−x2+x1=avg⇒x2=x1+a2−avg=xn+(a1+a2−2avg)第 3 个人：a3−x3+x2=avg⇒x3=x2+a3−avg=xn+(a1+a2+a3−3avg)\text{第 3 个人：} a_3 - x_3 + x_2 = avg \Rightarrow x_3 = x_2 + a_3 - avg = x_n + (a_1 + a_2 + a_3 - 3avg)第 3 个人：a3−x3+x2=avg⇒x3=x2+a3−avg=xn+(a1+a2+a3−3avg)⋯\cdots⋯第 i 个人：xi=xn+(∑j=1iaj−i⋅avg)\text{第 } i \text{ 个人：} x_i = x_n + \left(\sum_{j=1}^i a_j - i \cdot avg\right)第 i 个人：xi=xn+(j=1∑iaj−i⋅avg)

4. 简化公式

令 Ci=∑j=1iaj−i⋅avgC_i = \displaystyle\sum_{j=1}^i a_j - i \cdot avg Ci=j=1∑iaj−i⋅avg。

CiC_i Ci 是可以提前算出来的常数（即前 ii i 个小朋友初始糖果与目标总糖果的差值）。那么：

x1=xn+C1,x2=xn+C2,⋯ ,xn−1=xn+Cn−1x_1 = x_n + C_1, \quad x_2 = x_n + C_2, \quad \cdots, \quad x_{n-1} = x_n + C_{n-1}x1=xn+C1,x2=xn+C2,⋯,xn−1=xn+Cn−1xn=xn+Cn（由于总数不变，Cn 必然为 0）x_n = x_n + C_n \quad \text{（由于总数不变，} C_n \text{ 必然为 0）}xn=xn+Cn（由于总数不变，Cn 必然为 0）

5. 转化代价函数

题目要求的最小代价是传递糖果的总次数，即 ∑i=1n∣xi∣\displaystyle\sum_{i=1}^n |x_i| i=1∑n∣xi∣。

代入上面的简化公式：

代价=∣xn+C1∣+∣xn+C2∣+⋯+∣xn+Cn−1∣+∣xn+0∣\text{代价} = |x_n + C_1| + |x_n + C_2| + \cdots + |x_n + C_{n-1}| + |x_n + 0|代价=∣xn+C1∣+∣xn+C2∣+⋯+∣xn+Cn−1∣+∣xn+0∣

整理形式：

代价=∣xn−(−C1)∣+∣xn−(−C2)∣+⋯+∣xn−(−Cn−1)∣+∣xn−0∣\text{代价} = |x_n - (-C_1)| + |x_n - (-C_2)| + \cdots + |x_n - (-C_{n-1})| + |x_n - 0|代价=∣xn−(−C1)∣+∣xn−(−C2)∣+⋯+∣xn−(−Cn−1)∣+∣xn−0∣

关键点： 这个式子的物理含义是，在数轴上找一个点 xnx_n xn，使得它到点集 {0,−C1,−C2,…,−Cn−1}{0, -C_1, -C_2, \dots, -C_{n-1}} {0,−C1,−C2,…,−Cn−1} 的 距离之和最小。

根据经典数学结论：数轴上一个点到一组点距离之和最小，该点必然是这组点的中位数。

二、解题步骤总结

  1. 计算糖果平均值 avg；
  2. 计算数组 CC C 的各个元素：Ci=Ci−1+ai−avgC_i = C_{i-1} + a_i - avg Ci=Ci−1+ai−avg；
  3. 对数组 CC C 进行排序；
  4. 找到 CC C 的中位数 midmid mid；
  5. 累加计算每个 ∣Ci−mid∣|C_i - mid| ∣Ci−mid∣ 的绝对值，结果即为最小代价。

四、关键点解析（新手必看）

为什么是中位数？

想象你在一条路上放了几个车站，你想盖一间房子，让房子到所有车站的总路程最短。如果你把房子放在最左边，往右移动一点点，到左边车站的距离增加了，但到右边所有车站的距离都减少了。只要右边的车站比左边多，往右移就是赚的。直到左右车站数量相等时，你就达到了最优位置——这就是中位数的原理。

数据类型注意：

题目中 aia_i ai 可达 2×1092 \times 10^9 2×109，如果 n=106n = 10^6 n=106，糖果总数可达 2×10152 \times 10^{15} 2×1015，远超 int 的范围（约 21 亿）。因此必须使用 long long。

前缀和的妙用：

通过前缀和将”环形”的相互依赖关系转化为了独立的常数项，这是本题从”模拟/搜索”转变为”贪心/数学”的破局点。
